# 2017년 교회력
2017년 개신교(기독교대한감리회) 교회력

* 2017년 사순절은 재의 수요일인 3월 1일부터입니다.  
* 부활절은 춘분이 지난 첫번째 만월을 지나서 첫번째 주일입니다.  
* 부활절을 거꾸로 계산하여 주일을 제외한 40일을 뺀 날이 재의 수요일이 됩니다.
